# led_spiral_lamp

## Pinout

Data-Pin: 
- Port: 2

Button-Input:
- Port: 3
- Active: Low
- Debounced

## LED-Modes:

1. Rainbow
2. Spiral Bottom - Top
3. Random Colors
4. Rotated Rainbow
5. Top and Bottom to center
6. Advanced Spiral
7. Solid Red
8. Solid Orange
9. Solid Green
10. Solid Cyan
11. Solid Blue
12. Solid Magenta
